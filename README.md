# 🚗 Locatech - Sistema de Locadora de Veículos

## 📌 Sobre o Projeto
O **Locatech** é um sistema de gerenciamento para uma locadora de veículos, desenvolvido como parte da pós-graduação da FIAP.

O objetivo do projeto é aplicar conceitos de desenvolvimento backend utilizando Java com foco em boas práticas, organização de código e arquitetura em camadas.

O sistema permitirá o controle de operações essenciais como cadastro de veículos, clientes e gerenciamento de locações.

---

## 🛠️ Tecnologias Utilizadas

- Java
- Spring Boot
- JDBC (Java Database Connectivity)
- Arquitetura MVC (Model-View-Controller)

> ⚠️ Este projeto está em desenvolvimento e novas tecnologias poderão ser adicionadas futuramente.

---

## 🧱 Arquitetura

O projeto segue o padrão de arquitetura em camadas (MVC):

- **Model**: Representação das entidades do sistema  
- **View**: Camada de apresentação (API REST)  
- **Controller**: Responsável por receber e tratar as requisições  
- **Service**: Contém as regras de negócio  
- **Repository (DAO)**: Comunicação com o banco de dados via JDBC  

---

## ⚙️ Funcionalidades

- Cadastro de veículos
- Cadastro de clientes
- Registro de locações
- Consulta de veículos disponíveis
- Controle de devoluções

---

## 📖 Aprendizados
